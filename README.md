# Project 01 – OSM Data Cleaning & Topology QA/QC  
**Gueliz, Marrakech – Morocco**

## Objective
Prepare OpenStreetMap roads and buildings data for network analysis and urban planning by applying rigorous QA/QC and topology enforcement.

## Data
- OpenStreetMap (roads & buildings)
- Administrative boundary of Gueliz

## Workflow
1. CRS validation and reprojection
2. Attribute table cleaning
3. Geometry validation
4. Buildings topology rules (no overlaps, no multipart)
5. Road topology and network readiness
6. Dangling end classification
7. Pseudo-node evaluation
8. Final validation

## Outputs
- Clean road network ready for routing
- Clean building footprints
- Final cartographic map

## Tools
- QGIS
- Topology Checker
- OSM data
